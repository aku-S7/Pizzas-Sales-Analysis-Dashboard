Pizza Sales Analysis Dashboard

Title
Pizza Sales Analysis Dashboard: Interactive Insights for Pizza Sales Performance
A dynamic and interactive Power BI dashboard built to explore and analyze pizza sales data, focusing on revenue trends, product performance, and customer behavior.

Description
The Pizza Sales Analysis Dashboard is a visually engaging tool designed to help business owners and analysts explore pizza sales trends, identify top-selling products, and make data-driven decisions. It includes metrics like total revenue, peak sales periods, and product performance, enabling actionable insights for sales optimization and inventory planning.

Tech Stack
The dashboard was built using the following tools and technologies:
SQL – For querying, cleaning, and transforming raw sales data.
Power BI Desktop – Main platform for creating interactive reports and visualizations.
Power Query – Used for data transformation, reshaping, and preparation.
DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
Data Modeling – Relationships established among tables for accurate cross-filtering and aggregation.
File Formats – CSV/Excel used as source data for analysis and visualization.

Features:
Dynamic visualizations with filters and slicers for time periods, locations, and pizza types.
Sales trend analysis to identify peak sales periods and growth opportunities.
Top products & revenue analysis to pinpoint best-selling pizzas.
Customer segmentation for targeted marketing insights.
Interactive charts, graphs, and KPIs for quick decision-making.
